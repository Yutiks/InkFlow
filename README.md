# InkFlow

A modern tattoo studio website featuring online booking, artist portfolios, and an AI assistant.

## Tech Stack

### Frontend
- React
- Vite
- React Router

### Backend
- FastAPI
- PostgreSQL
- SQLAlchemy

## Features

- Home page
- Portfolio
- Artists page
- Online booking
- Admin panel
- AI recommendations

## Project Structure

frontend/
backend/
docs/

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```
